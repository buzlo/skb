# skb
CRM система на нативном JS. Получение списка клиентов из API и вывод в виде таблицы, сортировка по нескольким параметрам, поиск. Модальные окна для редактирования и удаления клиента. Для backend взято готовое решение

Для работы выполнить из папки project команду npm start и перейти по URL, указанному после "INFO Accepting connections at"
